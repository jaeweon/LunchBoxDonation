# 프로젝트 소개   
우리의 목표는 힘들게 살아가는 이웃들에게 따뜻한 식사를 제공하는 것입니다. 
서비스를 통해 구매한 도시락은 취약계층 시민들에게 기부됩니다. 
이렇게 구매한 도시락은 누구나 손쉽게 도움의 손길을 전달할 수 있는 수단이 됩니다.
